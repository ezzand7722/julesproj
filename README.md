# Student Flowchart & Mindmap Tool

A lightweight, web-based tool for creating flowcharts and mindmaps, optimized for students.

## Features
- **Drag-and-Drop Interface**: Easily add Process, Decision, Start/End, Node, Triangle, Star, and Hexagon shapes.
- **Connectors**: Draw lines between shapes with visible arrowheads.
- **Customization**: Change fill colors, stroke colors, stroke width, and text colors via the Property Panel.
- **Tools**: Undo/Redo, Zoom, Pan, and Export to SVG.
- **Auto-Save**: Your work is automatically saved to your browser's local storage.

## How to Use
1. Open `index.html` in your web browser.
2. Drag shapes from the left toolbar onto the canvas.
3. Use the "Connect" tool to link shapes.
4. Double-click any shape to edit its text.
5. Select a shape to modify its properties in the right sidebar.

## Project Structure
- `index.html`: Main application file.
- `style.css`: Stylesheet.
- `script.js`: Application logic.
